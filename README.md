# FinIQ 💸
### Your Personal Finance Mentor. Always On.

FinIQ is an AI-powered financial literacy platform built for India — 
designed for beginners, college students, and non-technical users who 
want to learn and practice finance in a safe environment.

---

## 🚀 Live Demo
[hubweavers-finiq.vercel.app](https://hubweavers-finiq.vercel.app)

---

## ✨ Features

- 📚 **Learn** — Structured finance courses for beginners
- 📖 **BookIQ** — Summaries of top finance books
- 📊 **Track** — Track your financial progress and habits
- 🏆 **Community** — Global leaderboard, streaks and rewards store
- 🤖 **SimZone** — Paper trading simulator with real NSE/BSE 
  data and an AI mentor that guides you in real time

---

## 🧱 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Vite |
| Chart | TradingView Lightweight Charts |
| Real-time Data | Yahoo Finance API (NSE/BSE) |
| AI Mentor | Gemini 1.5 Flash |
| Database | Firebase Firestore |
| Auth | Firebase Authentication |
| Hosting | Vercel |

## 🛠️ Local Setup

```bash
git clone https://github.com/srishantdhongade39-beep/hubweavers.git
cd hubweavers
npm install
npm run dev
```

> ⚠️ You will need to create your own `.env` file with 
> Firebase and Gemini API credentials to run this project locally.

---

## 🤖 SimZone Highlights

- Live candlestick charts for NSE stocks (RELIANCE, HDFC, TCS, INFY and more)
- Virtual balance of ₹1,00,000 to practice trading
- Real-time BUY / SELL execution with P&L tracking
- AI Mentor powered by Gemini that explains every trade in simple language
- Watchlist, Portfolio, and Orders tracking
- Built for complete beginners — no prior finance knowledge needed
